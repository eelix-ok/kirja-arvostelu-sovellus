# kirja-arvostelu-sovellus

Ominaisuudet

- Käyttäjä voi tehdä tunnuksen ja kirjautua sisään
- Käyttäjä voi lisätä kirjoja
- Käyttäjä voi muokata ja poistaa lisäämiään kirjoja
- Käyttäjä voi selata kaikkia kirjoja
- Käyttäjä voi etsiä kirjoja nimellä, kirjailijalla tai kategorialla
- Käyttäjäsivu näyttää käyttäjän lisäämät kirjat
- Kirjoille voidaan lisätä kategorioita
- Käyttäjät voivat kirjoittaa arvosteluja ja kommentteja kirjoihin
